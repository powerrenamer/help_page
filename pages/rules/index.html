Rule Usage Instructions

Rule Combinations

  • Multiple rules can be added and executed in sequence
  • Rule order can be adjusted by dragging
  • Each rule can be individually enabled or disabled

Rule Configuration

  • Each rule has its specific configuration options
  • Effects can be previewed in real-time during configuration
  • Rules can be saved as presets for future use
