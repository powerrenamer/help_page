Welcome to Power Renamer

Power Renamer is a powerful batch file renaming tool. This documentation will help you understand and use all its features.

Quick Start

  1. Click the "Add Files" button to select files to rename
  2. Click the "Add Rule" button to add renaming rules
  3. Adjust rules flexibly and preview renaming results
  4. Click the "Rename" button to execute the renaming operation
  5. Click the "Undo" button to restore original names
  6. Click the "Preset Management" side menu button to save and load rule combinations

Features

  • Support multiple renaming rules that can be freely combined
  • Real-time preview of renaming results
  • Support batch operations
  • Save preset configurations
  • Support undo operations

Tip

It is recommended to read the "Getting Started" guide first to understand basic operations, then refer to specific rule documentation as needed.
