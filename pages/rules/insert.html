Insert Rule

The insert text rule allows you to insert custom text at specific positions in filenames in various ways. Here's a detailed usage guide.

Feature Overview

  • Insert text at the beginning or end of filenames
  • Insert text at a specified character position in filenames
  • Insert text before or after specified target text in filenames
  • Replace entire filename with specific text
  • Option to ignore file extensions, ensuring insert operations don't affect file types
  • Insert metadata tags

Usage Steps

1. Set Text to Insert

In the "Insert Text" section, enter the text you want to insert. For example: _New

2. Choose Insert Position

Prefix Mode

Insert text at the beginning of filename

example.txt → _Newexample.txt

Suffix Mode

Insert text at the end of filename

example.txt → example_New.txt

Specific Position

Insert text at a specified character position in filename

example.txt → ex_Newample.txt (Insert at position 3)

Optional: Enable "Right to Left" to count position from the end of filename

Before Target Text

Insert text before specified target text

example.txt → ex_Newample.txt (Target text is "ample")

After Target Text

Insert text after specified target text

example.txt → ex_Newample.txt (Target text is "ex")

Replace Current Name

Replace entire filename with custom text

example.txt → NewFile.txt (Ignore extension enabled) example.txt → NewFile (Ignore extension disabled)

3. Configure Additional Options

Ignore Extension

When enabled, insert operation only affects filename, keeping file extension unchanged

example.txt → example_New.txt

Right to Left

When enabled in specific position mode, character position is counted from the end of filename

Important Notes

Target Text Matching

  • When using "Before Text" or "After Text" modes, ensure target text exists in the filename
  • Insert operation will not take effect if target text is not found

Valid Character Positions

  • In specific position mode, character position must be within filename length
  • Invalid positions will be ignored

Extension Handling

  • When "Ignore Extension" is enabled, insert operation won't affect file extensions
  • In replace current name mode, this option controls whether to preserve original extension
