Clean Up Rule

The clean up rule allows you to clean and standardize filenames by removing unwanted content and replacing characters. This guide will help you quickly understand how to configure and apply this rule.

Feature Overview

  • Remove content within different types of brackets
  • Clean excess spaces and punctuation
  • Remove special characters
  • Standardize spaces and separators
  • Support combining character processing

Usage Steps

1. Select Cleaning Options

Bracket Processing

Choose bracket types to process:

  • Parentheses ()
  • Square brackets []
  • Curly braces {}

Space Processing

Configure space-related options:

  • Remove leading and trailing spaces
  • Merge consecutive spaces
  • Add space before uppercase

Special Characters

Process special characters:

  • Remove combining characters, such as accent marks

2. Configure Processing Options

Protection Options

  • Protect extensions
  • Number sequences are not cleaned, such as version numbers: 1.0.0, and similar number sequences
