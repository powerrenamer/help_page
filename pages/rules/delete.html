Delete Rule

The delete rule allows you to remove specified text or characters from filenames. This rule provides multiple deletion modes and options to help you precisely control what to delete.

Feature Overview

  • Support deleting characters from specified delimiter
  • Support deleting characters from specified position
  • Support deleting specified number of characters
  • Support deleting until specified delimiter
  • Support deleting to end of file
  • Support right-to-left character deletion
  • Support preserving delimiters
  • Support ignoring extensions
  • Support deleting current name

Usage Steps

1. Position-Based Deletion

Left to Right Delete

Delete specified number of characters starting from specified position

IMG_1234.jpg → 1234.jpg (Delete 4 characters starting from position 1)

Right to Left Delete

Count from right side, delete specified number of characters at specified position (including the character at that position)

photo_2024_old.jpg → photo_2old.jpg (From 4th character '_' from right, delete 4 characters including '_', i.e., delete "024_")

Note: Deletion range includes the starting position character

Delete to End

Delete from specified position to the end of filename

photo_2024_old.jpg → photo.jpg (Delete from position 6 to end)

2. Delimiter-Based Deletion

Delete Specified Count

Delete specified number of characters starting from delimiter

photo_2024_old.jpg → photo2024_old.jpg (Delete 1 character starting from "_" delimiter)

Delete to Next Delimiter

Delete from one delimiter to another delimiter

photo_2024_old.jpg → photoold.jpg (Delete from first "_" to second "_")

Note: Assuming "Preserve Delimiters" is not selected

Preserve Delimiters Option

Option to keep delimiters when deleting content

photo_2024_old.jpg → photo__old.jpg (Delete content between delimiters but keep delimiters)

3. Special Options

Ignore Extension

When enabled, deletion operation won't affect file extension

photo.txt.jpg → photo.jpg (Protect final extension when deleting ".txt")

Delete Current Name

Delete current filename while keeping extension (if ignore extension is enabled)

photo_2024.jpg → .jpg (Delete entire filename but keep extension)

Important Notes

Delete Position

  • When using specified position mode, positions are counted starting from 1
  • For right-to-left deletion: first count from right to specified position, then delete specified number of characters including starting position
  • Example: In "example.txt", deleting 3 characters from 3rd position (p) from right will delete "amp" (including p itself)

Extension Handling

  • Recommended to enable "Ignore Extension" to protect file type
  • If processing extensions, ensure it won't affect file usability
