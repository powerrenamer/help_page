Strip Rule

The strip rule allows you to remove or keep specific types of characters in filenames.

Feature Overview

  • Support filtering multiple types of characters
  • Choose filtering position
  • Support keeping or removing matching characters
  • Option for case sensitivity
  • Option for extension handling

Filter Options

Filterable Character Types

English Letters

Includes all English letters a-z

abcdefghijklmnopqrstuvwxyz

Numbers

Includes all numbers 0-9

1234567890

Symbols

Common special symbols

!?@#$%^&~`_+-=

Brackets

Various types of brackets

(){}[]

Custom Text

Specify any characters to filter

Any user-defined combination of characters

Filter Position

Any Position

Find and process matching characters throughout the filename

file(1).txt → file.txt (remove brackets and numbers)

Start of Filename

Only process matching characters at the beginning of filename

123file.txt → file.txt (remove leading numbers)

End of Filename

Only process matching characters at the end of filename

file123.txt → file.txt (remove trailing numbers)

Advanced Options

Processing Mode

Remove Matching Characters

Delete all matching characters (default mode)

file123.txt → file.txt (remove numbers)

Keep Matching Characters

Keep only matching characters, delete everything else

file123.txt → 123.txt (keep only numbers)

Other Options

Case Sensitive

When enabled, uppercase and lowercase letters are treated as different characters

FileNAME.txt → Result depends on whether uppercase or lowercase letters are selected for filtering

Ignore Extension

When enabled, won't process the file extension part

file123.txt → file.txt (instead of file123txt)

Important Notes

Option Combinations

  • Multiple character types can be selected for filtering simultaneously
  • Keep matching characters mode changes the processing logic
  • Recommended to test effects on a small set of files first

File Extensions

  • Recommended to keep "Ignore Extension" option enabled
  • Be cautious when processing extensions
  • Changing extensions may make files unopenable

Processing Tips

  • Preview changes before applying
  • Backup important files
