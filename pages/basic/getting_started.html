Getting Started

This guide will help you quickly understand and use the basic features of Power Renamer.

Main Features

  • Batch rename files
  • Support multiple renaming rules
  • Real-time preview of renaming results
  • Support undo operations
  • Save current rule list as presets

Basic Usage Process

1. Select Files

Click the "Add Files" button in the toolbar to open the file selector and add files you want to rename:

2. Add Rules

Click the "Add Rule" button and select the renaming rules you need:

  • You can add multiple rules
  • Rules will be applied to files in order
  • You can drag rules to adjust their order
  • Click the settings icon to configure rule parameters
  • Click the delete icon to remove a rule

3. Preview Results

You can see the preview of renaming results in the file list:

  • Original filename is shown above
  • New filename is shown below
  • Warning indicators will appear if there are conflicts or errors

4. Execute Renaming

After confirming the preview results are correct, click the "Rename" button to execute the operation:

  • Progress will be shown during the renaming process
  • You can click "Undo" to restore original names (⚠️ only supports undoing the last rename operation)
  • You can save the current rule combination as a preset for future use

Usage Tips

Rule Combinations

  • Start with simple rules, then add complex ones gradually
  • You can enable and disable rules by toggling the checkbox in front of them
  • Pay attention to rule execution order
  • Use the preview feature to verify each step

Preset Management

  • Create presets for frequently used rule combinations
  • Give descriptive names to your presets
  • Regularly clean up unused presets

Safety Tips

  • Backup important files before renaming
  • Test on a small number of files before large batch operations
  • Don't close the application before using the undo feature
